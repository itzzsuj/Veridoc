# Veridoc - 5-Agent Document Forgery Detection System

AI-powered document verification for Indian academic marksheets (TNBSE, Anna University, CBSE)

## Features
- 5-Agent forensic analysis
- SHA-256 hash verification
- OpenCV visual forensics
- CNN seal/stamp verification
- Gemini + GPT-4o analysis
- Pinecone RAG matching

## Tech Stack
- React + Material UI
- FastAPI
- Firebase


## Installation
```bash
cd myapp
npm install
npm start
```
